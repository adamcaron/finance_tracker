An app to manage finances.

## Up and Running

`git clone git@github.com:adamcaron/finance_tracker.git`
`cd finance_tracker`
`npm install`
`touch personal.json`
Add to 'personal.json'
```
{
  "weeklyIncome": 100000000000 // your weekly income as a Number
}
```
Run it! `node app`
