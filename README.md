# Finance Tracker

## Up and Running

 - `git clone git@github.com:adamcaron/finance_tracker.git`
 - `cd finance_tracker`
 - ensure Docker is running

### dev

 - `bash dev-stack.sh`
 - navigate to yourhostname for app ui
 - navigate to yourhostname:8080 for proxy dashboard

### prod

 - `bash prod-stack.sh`
 - navigate to yourhostname for ui
